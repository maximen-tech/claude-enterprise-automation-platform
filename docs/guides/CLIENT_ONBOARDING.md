# Documentation Onboarding Client

Ce guide détaille toutes les étapes pour onboarder un client dans la plateforme miracle, automatiser l'analyse BI, activer les skills principaux, exporter le reporting personnalisé, et intégrer tous les workflows de support/monitoring/optimisation.

- Créer compte client
- Collecter site web/infos métier
- Lancer analyse BI via workflow
- Exporter rapport automatisé
- Onboard sur dashboard personnalisé
- Mettre en place monitoring web
- Automatisation support client (service-client)
- Plan d'intégration progressive des autres skills/agents
